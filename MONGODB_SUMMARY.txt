## Troubleshooting

**Q: Can't connect to MongoDB?**
A: Check if MongoDB is running. Try `docker run -d -p 27017:27017 mongo`

**Q: Port 5050 in use?**
A: Change PORT in `.env` or kill the process using the port

**Q: Changes not saving?**
A: Check browser console (F12) for errors, verify API_URL in script.js

**Q: MongoDB Atlas not working?**
A: Verify credentials in connection string, check IP whitelist
